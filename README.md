# STRAWBERRRY 🍓

A premium strawberry juice brand website with a bold, editorial aesthetic — dark backgrounds, oversized typography, and a product-first layout that feels more luxury fashion than typical food brand.

## ✨ Features

- Full-screen hero with product centerstage
- Editorial typography — large, bold, high contrast
- Dark luxury aesthetic with red accent palette
- Navigation with Products, Story, and Quality sections
- Clean, minimal layout — zero clutter

## 🛠 Tech Stack

- **React** — component-based UI
- **CSS / Tailwind** — custom styling and layout
- **Vercel** — deployment *(coming soon)*

## 🚀 Getting Started

```bash
git clone https://github.com/yourusername/strawberrry.git
cd strawberrry
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 🎨 Design Philosophy

Designed to feel like a premium lifestyle brand — not just a juice company. The dark editorial look is intentional: it positions the product as something aspirational rather than generic. Inspired by high-end beverage and fashion brand websites.
